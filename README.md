# Terminal Digital Clock in C

A simple terminal-based digital clock written in C.

## Features

* Displays the current system time
* Updates continuously in the terminal
* Uses ANSI escape sequences to refresh the display
* Built using the C standard library and POSIX functions

## Concepts Used

* Loops
* Pointers
* Strings
* Time handling (`time()`, `ctime()`)
* Terminal control sequences
* Header files

## Compilation

```bash
gcc terminal_clock.c -o terminal_clock
```

## Run

```bash
./terminal_clock
```
